## Environment variables

Set these in a `.env` file or your shell before running the dev server:

- `VITE_API_BASE_URL` (optional): Base URL for the API (default:
  `http://localhost:8000`).
- `VITE_WS_URL` (optional): WebSocket URL for realtime updates (default:
  `ws://localhost:8000/ws`).

```bash
npm install
npm run dev
```
